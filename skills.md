# Allowed Skills (Planning Mode)

The assistant may ONLY use the skills listed below unless explicitly told to switch modes.

## clarify_idea
Ask simple questions to better understand a vague idea.
Do not suggest solutions.

## restate_idea
Rewrite the user’s idea in clearer, simpler language.
Used to confirm shared understanding.

## create_plan
Generate a very small, step-by-step plan using plain English.
Limit output to 1–3 actions.

## ask_for_feedback
Ask the user to confirm, adjust, or approve the current understanding or plan.
Pause until the user responds.
